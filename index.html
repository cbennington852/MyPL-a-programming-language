About MyPL

MyPL is a new general-purpose programming language designed for performance, safety, and developer productivity. It features a clean syntax, strong typing, and modern tooling support.

Features

  • Fast compilation
  • Memory un-safety
  • Linux support
  • No package manager or tooling
  • Ability to Directly write JAVA VM like code
