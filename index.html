About MyPL

MyPL is a new general-purpose programming language designed for performance, productivity, and developer control. It features a clean syntax, strong typing, and direct control. MyPL emphasizes a minimal yet expressive core language, enabling developers to write concise and readable code without sacrificing power or flexibility.

Features

  • Fast compilation
  • Direct control over VM
  • Linux support
  • Automatic documentation generator

Documentation

println(value)

Description: Prints the given value followed by a newline character.

Parameters: value - The value to print.

println("Hello, world!")

print(value)

Description: Prints the given value without adding a newline.

Parameters: value - The value to print.

print("Hello")

str_val(value)

Description: Converts the given value (integer, float, etc.) to a string.

Returns: A string representation of the value.

str_val(42)     // "42"
str_val(3.14)   // "3.14"

int_val(value)

Description: Converts a string or number to an integer.

Returns: An integer value.

int_val("23")   // 23

dbl_val(value)

Description: Converts a string or number to a double-precision floating point number.

Returns: A double value.

dbl_val("3.25") // 3.25

size(string)

Description: Returns the number of characters in the string.

Returns: An integer length.

size("foo bar") // 7

get(index, string)

Description: Retrieves the character at the specified index in the string.

Parameters:

  • index - The zero-based character position.
  • string - The string to access.

Returns: A single-character string.

get(0, "foo")   // "f"
get(1, "foo")   // "o"

readln()

Description: Reads a full line of input from the user.

Returns: A string containing the user's input.

str = readln()
