About MyPL

MyPL is a new general-purpose programming language designed for performance, productivity, and developer control. It features a clean syntax, strong typing, and direct control. MyPL emphasizes a minimal yet expressive core language, enabling developers to write concise and readable code without sacrificing power or flexibility.

Features

  • Fast compilation
  • Direct control over VM
  • Linux support
  • Automatic documentation generator
